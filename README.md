# Trip Tracker

A personal Excel-based tracker built for fun to organize trip planning — keeping itinerary steps and checklists in one place so it's always clear what's next.

## Features

- Itinerary tracking for each stage of the trip
- Checklist system to mark progress and pending actions
- Simple, at-a-glance view of what to tackle next

## Why I Made This

Built as a personal project to keep trip planning organized without needing a complicated app — just a clean, functional spreadsheet that does the job.
